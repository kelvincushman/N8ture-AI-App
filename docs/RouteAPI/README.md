## Overview

The N8ture Route API is the backbone of the trail discovery and navigation features within the N8ture AI app. It provides services for searching trails, retrieving detailed trail information, planning custom routes, and tracking user journeys in real-time.

This API integrates deeply with the **Dynamic Difficulty System**, providing users with real-time, weather-adjusted difficulty ratings for trails.

### Key Features

-   **Trail Search:** Find trails by location, difficulty, length, and other criteria.
-   **Detailed Trail Data:** Access comprehensive information for thousands of trails, including maps, elevation profiles, and points of interest.
-   **Dynamic Difficulty:** Get real-time difficulty ratings based on weather, trail conditions, and user fitness.
-   **Route Planning:** Create and save custom hiking routes.
-   **Live Journey Tracking:** Record a user's path, distance, and elevation gain during a hike.
